# 📘 Assignment: Flask SQLite Web App

## 🎯 Objective

Build a simple web application with Flask and SQLite that stores data permanently and supports user input through HTML forms.

## 📝 Tasks

### 🛠️ Create the web application

#### Description

Implement a Flask app that renders HTML pages for viewing and adding records, and persists data in a SQLite database.

#### Requirements
Completed program should:

- Use Flask to define routes for displaying and submitting data
- Render HTML pages with forms and a list of stored records
- Store submission data in a SQLite database
- Show a confirmation or error message after form submission

### 🛠️ Add persistent storage and CRUD support

#### Description

Connect the Flask app to SQLite and implement at least create and read functionality for stored items.

#### Requirements
Completed program should:

- Create or open a SQLite database file
- Define a table for the data being stored
- Insert new records from form submissions
- Query the database to display saved records on the page
- Keep data available after the app restarts
