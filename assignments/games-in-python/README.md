
# 🎮 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a Hangman game using Python strings, loops, conditionals, and user input.

## 📝 Tasks

### 🛠️ Create the Hangman game

#### Description

Implement a text-based Hangman game that randomly chooses a hidden word and lets the player guess letters until they win or run out of attempts.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list
- Prompt the player to guess one letter at a time
- Display current word progress using underscores for unknown letters
- Track and show incorrect guesses remaining
- End the game with a win message when the word is fully guessed
- End the game with a lose message when attempts run out

### 🛠️ Add replay support

#### Description

Allow the player to start a new game after one round ends without restarting the program.

#### Requirements
Completed program should:

- Ask the player whether they want to play again after win or loss
- Reset the game state for a new round when the player chooses yes
- Exit cleanly when the player chooses no
